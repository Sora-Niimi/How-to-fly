## 🔧 自行构建 PDF

需要 Python 3.7+ 和 Edge/Chrome 浏览器。

```bash
# 1. 安装依赖
pip install markdown

# 2. 生成 HTML
cd scripts
python md2pdf.py --html

# 3. 通过 Edge headless 生成 PDF（Windows）
powershell -ExecutionPolicy Bypass -File md2pdf_headless.ps1
```

或者直接在浏览器中打开生成的 HTML 文件，Ctrl+P → 另存为 PDF。
